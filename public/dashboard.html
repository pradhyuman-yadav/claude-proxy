Supported Parameters
This proxy runs on Claude Code CLI, not the direct Anthropic API. Unsupported params are silently stripped before forwarding.
ParameterSupportNotes
model✓ yesAuto-normalized to canonical ID
messages✓ yessystem / user / assistant roles
max_tokens✓ yes
temperature✓ yes0 – 1

How to Use
Base URL {{baseUrl}}/v1
api_key {{apiKeyNote}}
curl
curl -X POST {{baseUrl}}/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer {{apiKeyValue}}" \
  -d '{
    "model": "claude-sonnet-4",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'
Python (openai SDK)
from openai import OpenAI

client = OpenAI(
    base_url="{{baseUrl}}/v1",
    api_key="{{apiKeyValue}}",
)
resp = client.chat.completions.create(
    model="claude-sonnet-4",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(resp.choices[0].message.content)
